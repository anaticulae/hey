# Example - footer

To generate the test data run update.sh.

The test data is based on two executions. On the one hand it is generated by
default parameterization and on the other hand there is the `oneline` prefixed
data. These data interprets the pdf as one horizontal line and is required for
detecting tables like table of content or table of figure.

```[bash]
rm rawmaker__*

rawmaker -i restructuredtext.pdf -o . --prefix=oneline --text
--char_margin=100.0

rawmaker -i restructuredtext.pdf -o . --toc --text --char_margin=5.0 --border
--boxes --font
```
