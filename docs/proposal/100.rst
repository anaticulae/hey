FP 100: Feature gradient
========================

The feature gradient describe the change of a feature belong to page
position from top to down.

There a different feature-gradients:

* text position
* text width
* text bounding area
* text distance
* font size
* horizontal lines / width / height
* image width / height

Table
-----

How to identify a table:

* horizontal line gradient
* text gradient

Footer
------

* Text gradient
* Horizontal line gradient
* Area difference
